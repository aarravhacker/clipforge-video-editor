Basic:
  - Brightness        (slider: -1 to 1)
  - Contrast          (slider: 0.5 to 2)
  - Saturation        (slider: 0 to 3)
  - Speed             (slider: 0.25x to 4x)
  - Volume            (slider: 0 to 2)

Blur Types:
  - Blur              (box blur, slider: 0-20)
  - Gaussian Blur     (gblur, slider: 0-50)
  - Motion Blur       (convolution-based)
  - Lens Blur         (lenscorrection)

Color Effects:
  - Grayscale (B&W)
  - Sepia
  - Vintage
  - Warm tone
  - Cool tone
  - Invert

Color Grading Presets (LUT):
  - Cinematic
  - LUT Warm
  - LUT Cold
  - B&W High Contrast
  - Bleach Bypass
  - Cross Process

Stylized:
  - Glitch            (RGB shift + hue oscillation)
  - Neon Glow         (edge detect + negate)
  - Emboss            (convolution kernel)
  - Pixelate          (downscale + upscale)
  - RGB Split         (chromatic aberration)
  - Film Grain        (temporal noise)
  - Vignette

Transform:
  - Mirror (horizontal flip)
  - Flip (vertical flip)
  - Kaleidoscope (hflip+vflip+hstack+vstack)

Advanced:
  - Chroma Key        (green screen removal)
  - Color Key         (specific color removal)

PRO FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  - Ken Burns zoom       (slow zoom in animation)
  - Audio volume control (0 to 2x)
  - Audio waveform       (visual waveform display)
  - Keyframe animations  (position, scale, rotation, opacity)
  - Watermark support

MCP SERVER (AI Control)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  - upload_video        - Upload video from disk
  - list_projects       - List all projects
  - get_timeline        - Get timeline state
  - trim                - Trim a clip
  - split_clip          - Split clip at time
  - apply_effect        - Apply any of 30 effects
  - add_transition      - Add any of 11 transitions
  - add_text            - Add text overlay
  - add_sticker         - Add emoji sticker
  - set_speed           - Set clip speed
  - set_speed_ramp      - Set variable speed segments
  - reverse_clip        - Toggle reverse
  - set_volume          - Set audio volume
  - set_zoom            - Toggle Ken Burns zoom
  - set_keyframe        - Set animation keyframe
  - export_video        - Render final video

HOW TO USE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Run: npm start
  2. Open: http://localhost:3040
  3. Upload a video (drag & drop or browse)
  4. Apply effects, transitions, text, stickers
  5. Preview in real-time
  6. Export final video

MCP (AI Control):
  1. MCP server runs on port 3001
  2. Send JSON-RPC requests to control editor
  3. Example: add effects, transitions, export
